# Virtual Server Monitoring and Performance Optimization

<img width="1371" height="394" alt="image" src="https://github.com/user-attachments/assets/d940eb8d-7adc-4b3d-bfe1-143da07bf3ad" />




This project implements an end-to-end server monitoring data pipeline that ingests raw server logs, processes them through multiple transformation layers, and visualizes the insights through an interactive dashboard.

Due to the lack of access to cloud infrastructure, the pipeline was implemented locally using Microsoft SQL Server and Power BI, while still following the modern Bronze–Silver–Gold Medallion Architecture used in cloud data platforms.
